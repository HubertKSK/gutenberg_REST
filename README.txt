Zadanie - "Analiza tekstu z projektu Gutenberg - REST API"
Korzystając z języka programowania Python i dostępnych bibliotek utwórz REST API pozwalające na:
1. Zliczanie wskazanego słowa w wybranej książce dostępnej w ramach projektu Gutenberg.
1. API powinno przyjmować (1) link do książki w projekcie Gutenberg w wybranym formacie (przykładowy link: "https://www.gutenberg.org/files/2701/2701-0.txt") i (2) wybrane słowo.
2. Wskazanie wszystkich słów występujących w treści książki wraz z ilością wystąpień.
3. Wskazanie dziesięciu najczęściej występujących słów w treści książki wraz z ilością ich wystąpień.
4. "Freestyle" - wykaż się kreatywnością. Zaproponuj i zrealizuj funkcjonalność API Twojego pomysłu. Wybór uzasadnij.
